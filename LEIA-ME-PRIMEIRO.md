# Arquivos atualizados para o perfil `pedvrezz`

Envie para a raiz do repositório `pedvrezz/pedvrezz`:

- `README.md`
- a pasta `assets`
- a pasta oculta `.github`

A pasta `.github/workflows/snake.yml` é obrigatória para a animação da cobra aparecer na aba Actions.

Depois do envio:

1. Abra a aba **Actions**.
2. Selecione **Generate contribution snake**.
3. Clique em **Run workflow**.
4. Aguarde a execução terminar.
5. Atualize a página do perfil.

O arquivo `SETUP.md` contém as instruções completas.
